# Images Directory

This directory contains legacy assets from the original template.

## Current Implementation

The diabetes prediction website uses:
- **CSS gradients** for visual backgrounds
- **Lucide React icons** for feature illustrations
- **Colored circles with initials** for testimonials
- **Icon-based graphics** instead of image files

## Adding New Images

If you want to add custom health-themed illustrations:
1. Place image files in this directory
2. Update components to reference them (e.g., `/images/your-image.png`)
3. Use modern formats (WebP, SVG) for better performance
